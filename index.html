ABC RWD Case Study

How might the ABC provide a better user experience to its website users?

The ABC website provides the Australian community with a vast range of media content and its current navigation can be better designed in order to provide users with an improved experience with consistent branding and structure across all interfaces.

Research


  • Heuristic & UI Evaluations
    Heuristic evaluations of the ABC homepage, news page, and navigation was conducted after establishing a typical user flow to better understand the strengths and weaknesses of the interface. UI analysis revealed shortcomings regarding minimalist design and user control- the interface was too busy and flooded with information and navigation could be improved with streamlined CTAs.
  • User Interviews
    Five regular users of the ABC website were interviewed. Research questions focused on the user flow and ease of locating information and key CTAs on the website. User feedback was focused on:
    • the difficulty of the sign up process and cluttered interface
    • the difficulty in locating local news stories and,
    • preference to use the search bar over navigation which indicated issues with its usability.
  • User Persona
    Using the data collected in our user interviews we were able to distill motivations and needs into an empathy map that better reflected a typical user to continue guiding our design in the right direction.

Problem Statement

The ABC website provides the Australian community with a vast range of media content. Each branch of the ABC that is featured on the website (ABC iView, Kids, Radio, News etc) has its own navigation, structure and branding elements. The result leaves the website feeling incohesive and cluttered. The homepage its cramped and navigating back to it can be difficult as it is not clearly signposted. How might we improve in the interface of the ABC webite to better serve users?

Prototyping

User Testing and Feedback

User testing revealed some issues sound accessibility, aesthetics and consistency

  • There needed to be a way to collapse the dropdown menu instead of relying on the ‘back’ button
  • Buttons needed to be more uniform, some are different sizes
  • Hamburger menu should lose the circle with drop shadow- it would look cleaner just the hamburger lines on white background
  • Change text to black on the yellow, light blue, orange navigation tabs- they all failed accessibility contrast tests
  • The red tab needs to be altered as it doesn’t pass accessibility standards and black text won’t look good
